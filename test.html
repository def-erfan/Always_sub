SVG Background Example

This page demonstrates using an SVG image as a background for the body element. The diagonal line pattern you see in the background is an SVG created directly in the CSS using a data URL.

External SVG File

Reference an external .svg file:

background-image: url('pattern.svg');

Inline SVG (Data URL)

Embed the SVG directly in CSS:

background-image: url('data:image/svg+xml;utf8,<svg...>...</svg>');

CSS Background Properties

Control how the SVG displays:

background-size: cover;
background-repeat: no-repeat;
background-position: center;

Tips for Using SVG Backgrounds

  • SVGs scale perfectly at any size without quality loss
  • Use simple patterns for better performance
  • You can manipulate SVG colors with CSS if using inline SVG
  • Test contrast between background and content
  • Consider using semi-transparent SVGs over solid colors
